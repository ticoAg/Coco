# AgentMesh

AgentMesh 是一个围绕 **Codex CLI** 的本地协作执行/落盘项目（WIP）：把一次开发任务沉淀为可复盘的任务目录（Task Directory：事件流、产物、人工介入点），并用多个 Codex worker 并行完成子任务。

> 当前状态：WIP（以设计与产物/运行时落地为主）
